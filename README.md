# HR Analytics Dashboard (Excel)

## Project Overview
This project presents an interactive HR Analytics Dashboard built in Microsoft Excel to analyze employee attrition, workforce distribution, and key HR metrics.

## Key Features
- Employee Attrition Analysis
- Department-wise Attrition Rate
- Attrition by Job Role
- Employee Age Distribution
- Business Travel Distribution
- Interactive Slicers and Filters
- KPI Tracking and Visualization

## key insights 
- Overall attrition rate :16.12%
- The 26–35 age group exhibited the highest attrition concentration.
- Laboratory Technicians recorded the highest attrition among job roles.
- Business travel patterns showed a higher concentration of employees in the Non-Travel category.
